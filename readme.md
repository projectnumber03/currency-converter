## Build commands:
1. Run your console.
2. Enter the root directory of the project.
3. Execute command: `mvn clean install`

## Run commands:
1. Run your console.
2. Enter this path: `/currencyconverter/target`
3. Execute command: `java -jar currencyconverter-1.0.jar`
